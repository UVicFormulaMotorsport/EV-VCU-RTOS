enum \mbox{\hyperlink{rb__tree_8h_ae7a3c3e2ca387d6507c8c8e61e3f87b5}{rbtraversal}}}



Evil traversal method specifier for traversing the tree. 

Function that applies some function to a subtree.


\begin{DoxyParams}{Parameters}
{\em rbt} & Pointer to an the {\ttfamily rbtree} you wish to apply the functions to\\
\hline
{\em node} & Pointer to the node that is the root of the tree you wish to apply functions to. This can be a subtree of another tree if needed\\
\hline
{\em func} & The function you would like to apply to the nodes. This takes in two parameters. The first is a pointer to the data of the node. The second parameter is a pointer to the shared cookie, which allows information to be preserved between different calls to {\ttfamily func}. This function returns an integer value, with 0 being ok, and the others being various error states.\\
\hline
{\em cookie} & This is a pointer to some space in memory that all of the calls to @func share, to preserve information.\\
\hline
{\em order} & This is a member of that specifies the order in which the tree will be traversed.\\
\hline
\end{DoxyParams}

DANGER!! RECURSION!! Beware of stack memory usage, since most tasks are memory limited.
